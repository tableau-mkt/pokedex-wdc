# Pokedex-Web-Data-Connector
A Tableau Web Data Connector for the Poke API (https://pokeapi.co/docsv2/#)

## Initial set-up
- NPM dependencies
  - `npm install`

## Usage
- Run `grunt` from the command line.
- Open Tableau, navigate to Web Data Connector
- Browse to http://localhost:9001
- Click 'Get Data'
